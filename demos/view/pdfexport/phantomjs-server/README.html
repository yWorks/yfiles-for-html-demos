yFiles for HTML PhantomJS Server

Prerequisites

To start the server, first download PhantomJS. Due to a regression in the PhantomJS 2.0 development, it is not possible to encode the server response as binary (see also Issue 13026).

For this application to work, PhantomJS 2.0.1 is the minimum requirement.

Running the PhantomJS Server

To run the PhantomJS Server, copy the PhantomJS executable to ./phantomjs-server/

Afterwards start the server from this directory with the following statement 

> phantomjs.exe server.js

This starts a server that is listening on http://localhost:8081.
