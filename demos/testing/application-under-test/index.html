Simple Testable App

This demo is a simple app that is used as the test candidate in the following testing demos:

  • Cypress Demo
  • Playwright Demo
  • Selenium WebDriver Demo

Please refer to the README files in the aforementioned demos for further guidance.

A simple way to make a yFiles App testable is making the GraphComponent available to the testing framework. In this app, this is achieved by setting the instantiated GraphComponent as a property of the Window object.
