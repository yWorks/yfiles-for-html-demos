# Vue Component Node Style Demo

This demo presents the Vue.js component node style that uses a [Vue.js](https://vuejs.org/) component to visualize nodes.

To start the demo

1.  Go to the demo's directory `demos-js/style/vue-component-node-style`.
2.  Run `npm install`.
3.  Run `npm run dev`.

This will compile the app and launch a dev server that will serve the demo application. Source code changes will be compiled and displayed automatically.
