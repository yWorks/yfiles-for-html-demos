Theming Demo

This demo shows various interaction visualization themes simultaneously.

Each graph component uses a different theme variant:

Top-left
Simple round
Top-right
Simple square
Bottom-left
Classic with non-default colors
Bottom-right
Classic with default colors

Things to Try

  • Navigate/edit one graph component to see the changes mirrored in the other components.
  • Choose a different color palette to use as the theme colors.
