This demo presents the React Component node and label style that leverages JSX and the powerful data binding features of the [React framework](https://reactjs.org/).

To start the demo

1.  Go to the demo's directory `demos-ts/style/react-template-node-style`.
2.  Run `npm install`.
3.  Run `npm run dev`.

This will compile the app and launch a dev server that will serve the demo application. Source code changes will be compiled and displayed automatically.
