Description

This demo shows how to create a React Component that renders the yFiles yfiles.view.GraphComponent.

React component properties and states are used in order to interact with the GraphComponent programmatically.

Building the demo

The demo is built using NPM:

> npm install
> npm run build

The build process consists of the following steps:

  • Transform the JSX sources in the ./src folder to plain JavaScript.
  • Bundle the transformed sources as well as the referenced yFiles library and demo modules into the ./dist/bundle.js file referenced by this page.

Things to Try

  • Toggle editing support using the "Toggle Editing" button.
  • After the graph has been edited, click to reload the initial sample graph.
  • Have a look at the JSX sources in the ./src folder to see how react state and properties are used to implement the demo functionality.
  • To play around with the React components, run npm run build:watch and edit the JSX files. The watch process will recreate the app bundle when changes are detected in the ./src folder.
