yFiles for HTML React Demo

The React Demo shows the integration of yFiles for HTML with the React user interface library.

To start the demo

  1. Go to the demo's directory demos/toolkit/react.
  2. Run npm install.
  3. Run npm run build.

The build process consists of the following steps:

  • Transform the JSX sources in the ./src folder to plain JavaScript.
  • Bundle the transformed sources as well as the referenced yFiles library and demo modules into the ./dist/bundle.js file referenced by this page.

To auto-update the bundle on code changes, run npm run build:watch.
