Description

This demo shows the integration of yFiles for HTML with TypeScript.

To start the demo please run npm install.

A TypeScript type definition file for the yFiles library is included in the distribution: ide-support/yfiles-api.d.ts. Besides from referencing this file, no special setup is required to use TypeScript with the yFiles library.

Things to See

It is common during yFiles for HTML development to extend existing functionality by inheriting from a yFiles class or implementing yFiles interfaces. This demo shows code examples for the most important cases:

  1. MyNodeStyle.ts shows how to inherit from a single yFiles class.
  2. MyMarqueeTemplate.ts shows how to implement a single yFiles interface.
  3. MyArrow.ts shows how to implement multiple yFiles interfaces.
