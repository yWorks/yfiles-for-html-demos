Description

This demo shows the integration of yFiles for HTML with TypeScript.

To start the demo please run npm install.

Several TypeScript type definition files for the yFiles library are included in the distribution:

ide-support/yfiles-api-umd-webstorm.d.ts
Typings for the UMD module version of the library, using HTML in the source code documentation.
ide-support/yfiles-api-es6-modules-webstorm.d.ts
Typings for the ES6 module version of the library, using HTML in the source code documentation.
ide-support/yfiles-api-umd-vscode.d.ts
Typings for the UMD module version of the library, using markdown in the source code documentation.
ide-support/yfiles-api-es6-modules-vscode.d.ts
Typings for the ES6 module version of the library, using markdown in the source code documentation.

Besides from referencing the appropriate file, no special setup is required to use TypeScript with the yFiles library.

Things to See

It is common during yFiles for HTML development to extend existing functionality by inheriting from a yFiles class or implementing yFiles interfaces. This demo shows code examples for the most important cases:

  1. MyNodeStyle.ts shows how to inherit from a single yFiles class.
  2. MyMarqueeTemplate.ts shows how to implement a single yFiles interface.
  3. MyArrow.ts shows how to implement multiple yFiles interfaces.
