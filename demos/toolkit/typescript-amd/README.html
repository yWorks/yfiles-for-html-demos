yFiles for HTML Typescript Demo

The TypeScript Demo shows the integration of yFiles for HTML with TypeScript.

To start the demo

  1. Go to the demo's directory demos/toolkit/typescript-amd.
  2. Run npm install.
  3. Open the index.html file.
