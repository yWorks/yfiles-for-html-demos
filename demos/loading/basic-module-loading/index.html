Creating the View

This demo is a minimal example of using yFiles for HTML to display a graph on a web page. More information about building a first yFiles for HTML application can be found in the developer's guide.

The demo code introduces class GraphComponent, which is the central UI element for working with graphs.

The graph consists only of a single node with a label as an example. The application does not provide any interactive features except mouse wheel zooming, which is enabled by default.
