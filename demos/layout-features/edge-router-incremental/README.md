# Incremental Edge Router

<img src="../../../doc/demo-thumbnails/layout-edge-router-incremental.webp" alt="demo-thumbnail" height="320"/>

[You can also run this demo online](https://www.yfiles.com/demos/layout-features/edge-router-incremental/).

This demo shows how to run the edge router algorithm on a predefined subset of edges in a graph.

## Setup

To achieve this, the [EdgeRouterData](https://docs.yworks.com/yfileshtml/api/EdgeRouterData)'s [scope](https://docs.yworks.com/yfileshtml/api/EdgeRouterData#scope) property has to be set to the subset of edges.

In this demo, the algorithm works on the subset of orange edges only.

## Things to Try

Click the button in the toolbar to run the routing algorithm.
