Hierarchical Layout with Layer Constraints

This demo shows how to customize the assignment of nodes to layers (layering) when using HierarchicalLayout.

Layer constraints

  • Node 9 is placed in the topmost layer with the placeAtTop method.

  • Node 7 is placed in the bottommost layer with the placeAtBottom method.

  • Node 0 is placed at least one layer below node 9 with the placeInOrder method.

  • Node 2 is placed in the same layer as node 0 with the placeInSameLayer method.

Things to Try

Click the button in the toolbar to toggle between hierarchical layout with and without configured constraints.
