Tree Layout with Subtree Placers

This demo shows how to use subtree placers with the TreeLayout. A subtree placer is responsible for the arrangement of a local root node and all of its subtrees.

yFiles comes with predefined subtree placers that provide a variety of subtree arrangement schemes. This example uses three of them:

  • Node 1 uses a LevelAlignedSubtreePlacer, that places child nodes with the same depth in the tree in the same horizontal layer.
  • Node 2 uses a LeftRightSubtreePlacer, that places child nodes left and right of a single vertical bus.
  • Node 6 uses a DoubleLayerSubtreePlacer, that places child nodes in two horizontal lines.
