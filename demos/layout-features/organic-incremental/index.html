Incremental Organic Layout

This demo shows how to run the OrganicLayout algorithm on a predefined subset of nodes in a graph.

To achieve this, two setup steps are necessary:

  • The algorithm has to know which set of nodes to work on. Class OrganicLayoutData offers the property scope, in which the set of nodes can be specified.

  • The layout algorithm has to be told whether a node and its surrounding nodes could possibly be laid out. To do so, the scopeModes property can be set to the following scope modes: AFFECTED, FIXED, INCLUDE_CLOSE_NODES or INCLUDE_EXTENDED_NEIGHBORHOOD.

In this demo, the layout algorithm is only applied to the orange nodes — for which there is no predefined layout information in the JSON sample data — and their surrounding nodes.
