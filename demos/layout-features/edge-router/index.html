Edge Router

This demo shows common configuration options for the EdgeRouter algorithm.

It highlights the configuration of various aspects, including:

  • Minimum distance settings like the edge distance and the node-to-edge distance.
  • Scope: the orange edges (fixed edges) are not routed by the algorithm. All other edges (affected edges) are routed.
  • Routing style: the blue edges are routed using the octilinear style, the others routed orthogonally.
  • Grouping: the pink edges are grouped at their target side.
  • Restricting ports via port candidates: at nodes '5' and '7', ports are restricted to be on their left or right side.
  • Grid routing: edges are routed on a 10x10 grid.
