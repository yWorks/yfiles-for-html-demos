Bus-Style Edge Routing

This demo shows how to configure the EdgeRouter to generate orthogonal bus-style routes. A bus is a shared segment among multiple edges, with nodes connecting to it via shorter segments.

Buses are defined using the add method of the buses property, which returns a configurable object for edge assignment.

The EdgeRouterBusDescriptor used in the add method offers customizable bus parameters.

Automatic Backbone Computation

In this demo, a default bus descriptor automatically determines the backbone segments for optimal route efficiency.
