Border-Aligned Label Model Demo

This demo shows how to implement a custom node label model to position labels around the circumference of a node.

Things to Try

  • Continuous Movement: Drag the label on the left node to observe how it moves continuously around the node's border
  • Snapping: Drag the label on the right node to see how it snaps to preferred discrete positions.
