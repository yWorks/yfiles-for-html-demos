Description

This demo shows how resizing of nodes can be restricted.

This is done with custom implementations of the interface INodeSizeConstraintProvider that are added to the lookup of the nodes.

Things to Try

Select and resize the nodes.

Node Types

  • Blue nodes cannot shrink. Additionally, neither side can become larger than three times its initial size in each resizing operation.
  • The minimum size of a green node is the size of its label. It gets never smaller than that.
  • The orange node must always encompass the black rectangle. Additionally, it has predefined minimum and maximum sizes.
