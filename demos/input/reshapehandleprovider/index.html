Description

This demo shows how resizing of nodes can be customized.

This is done with custom implementations of the interface yfiles.input.IReshapeHandleProvider that are added to the lookup of the nodes.

Things to Try

Select and resize the nodes.

Node Types

  • Red nodes cannot be resized. They don't display resize handles.
  • Green nodes show only four handles at the corners. During resizing, these handles maintain the aspect ratio of the node.
  • Orange nodes cannot extend beyond the black rectangle. Note that orange nodes can only be resized so that the new bounding box stays withing the enclosing black rectangle.
  • Blue nodes combine the behavior of the orange and green nodes.
