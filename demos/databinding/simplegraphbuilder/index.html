Description

This demo uses yfiles.binding.GraphBuilder or yfiles.binding.TreeBuilder to automatically build a graph from business data.
The business data can be stored in arrays or in JSON format.
The nodes are visualized by a yfiles.styles.TemplateNodeStyle instance that binds to the business data objects which yfiles.binding.GraphBuilder stores in the graph item's yfiles.graph.ITagOwner#tag.

  • nodesSource stores the business objects that represent the nodes.
  • edgesSource stores the business objects that represent the edges that connect the nodes.
  • groupsSource stores the business objects that represent the group nodes.

The files graph-builder-data.js, tree-builder-array-data.js, tree-builder-id-array-data.js and tree-builder-json-data.js show how the data has to be specified for the classes yfiles.binding.GraphBuilder and yfiles.binding.TreeBuilder.
