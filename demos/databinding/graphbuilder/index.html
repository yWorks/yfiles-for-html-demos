Description

This demo uses class GraphBuilder to interactively build and modify a graph from business data.

GraphBuilder stores the business data object of each node in the node's ITagOwner#tag property, thus, the properties of a node's TemplateNodeStyle can be bound to this object.

Things to Try

  • Select a sample in the combo box and inspect its data in the settings that define the graph.
  • Add or modify nodes in the Nodes Source.
  • Add or modify edges in the Edges Source.
  • Modify the SVG code for the Node Template.
  • Build a graph for your own data and bindings.

Graph Source Settings

  • While New Graph creates a new graph from scratch, Update Graph adjusts the graph to changed data and keeps nodes and edges whose source objects are still present.
  • Nodes Source contains the data that specifies the graph's nodes. This can either be an array or another object containing the node business data.
  • Edges Source contains the data that specifies the edges between nodes. This can also be an array or another object. In order to identify the nodes, the source and target node binding must be set.
  • The Source (Target) Node Binding is required to identify the source (target) node of the edge. This can either be the path to the property specifying the node's id, or a function that returns the node id.
  • If Lazy Node Definition is checked, the graph can contain nodes for objects that are specified in Edges Source but not Nodes Source.
  • The Node ID Binding is an optional property to specify a property path that returns a node's ID. If present, the node ID has to be used in the source and target node bindings.
  • The Edge Label Binding is an optional property that provides a label for an edge object.
  • Node Template defines the SVG template that is used to visualize a node. This may contain dynamic bindings as demonstrated in the samples that allow visualizing arbitrary properties of the node's business data, such as the name or id properties.

Since this demo evaluates the complete data source texts every time a button is clicked, Update Graph works as expected only for nodes source objects of type string. Note that this restriction applies only to the demo, not the class GraphBuilder.
